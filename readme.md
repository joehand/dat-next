## Install:

```

```

## Usage

Share `dir`:

```
dat-next <dir> [options]
    --sleep,-s     persist sleep files in .dat folder
```

Download `key` to `dir`:

```
dat-next <key> <dir> [options]
    --sleep,-s     persist sleep files in .dat folder
```
